class FlxEase

package flixel.tweens

@:directlyUsed

Static class with useful easer functions that can be used by Tweens.

Operation of in/out easers:

in(t)

return t;

out(t)

	return 1 - in(1 - t);

inOut(t)

	return (t <= .5) ? in(t * 2) / 2 : out(t * 2 - 1) / 2 + .5;
